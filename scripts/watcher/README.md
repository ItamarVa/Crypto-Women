# Blog watcher — קבצי הפעלה

בקרה על ה-watcher שממיר מסמכים מהתיקייה `D:\AI Projects\Crypto Women\Blog - Crypto Women`
לרשומות בלוג באתר (ראה `../blog-import.mjs`).

## הקבצים

| קובץ | מה עושה |
|------|---------|
| **install-autostart.bat** | מתקין הפעלה אוטומטית **בכל הדלקת מחשב, ללא צורך בלוגין**. הרץ **פעם אחת** (יבקש רק הרשאת מנהל — **בלי סיסמה**, logon מסוג S4U). מפעיל את ה-watcher גם מיד. |
| **uninstall-autostart.bat** | עוצר ומסיר את ההפעלה האוטומטית. |
| **start-watcher.bat** | מפעיל ידנית בחלון (מציג לוגים חיים). סגירת החלון = עצירה. שימושי לבדיקה. |
| **stop-watcher.bat** | עוצר watcher שרץ (ידני או מהמשימה). לא מסיר את ההפעלה האוטומטית. |

## הפעלה חד-פעמית
לחיצה כפולה על **install-autostart.bat** → אישור הרשאת מנהל → הקלדת סיסמת Windows. זהו.
מכאן כל מסמך שיישמר בתת-תיקיית קטגוריה יומר אוטומטית, יתווסף לבלוג, וייעלה לאתר.

## קטגוריות = תת-תיקיות
שמור את המסמך בתת-תיקייה בשם מדויק: `מיינדסט` · `השקעות` · `ביטקוין` · `בלוקצ'יין` · `מדריך` · `כללי`.
מסמך בשורש התיקייה → קטגוריה `כללי`.

## דרישות (כבר מותקנות במחשב זה)
`node`, `git`, `uv` + `markitdown` (הותקן דרך `uv tool install "markitdown[all]"`).

## הערות
- מסמכי **Google Docs** מקומיים (`.gdoc`) הם קיצור-דרך בלבד — יש לייצא ל-`.docx`/`.pdf` לתיקייה.
- הפרסום אוטומטי ומיידי (commit + push → בנייה → אונליין). כדי לעצור זמנית: `stop-watcher.bat`.
